# AI 智能摄影构图助手 (AI Camera K1)

> **基于进迭时空 K1 RISC‑V 板的实时 AI 取景器，自动识别场景，叠加构图指导字幕，帮助你拍出更美的照片。**
> 
> 全离线运行，支持物理按键、开机自启动、全屏显示，完美适配嵌入式竞赛与产品原型。

![demo](docs/images/demo.jpg)

---

## ✨ 功能亮点

- **🎯 实时目标检测**：YOLOv8n 320×320，可识别 80 类常见物体（人、车、猫、狗、杯子等）。
- **🏞️ 智能场景识别**：Places365 预训练模型（ResNet18），自动判断室内、海滩、城市、夜景、山脉等 10+ 种摄影场景。
- **📐 多维度构图指导**：
  - 三分法主体偏移分析
  - 水平线倾斜检测
  - 人像头顶留白评估
  - 画面拥挤度 / 对称度 / 色彩平衡
  - 黄金螺旋参考线（按 **G** 键切换）
- **💬 个性化的 AI 字幕提示**：中文自然语言提示，可扩展的 JSON 语料库，针对具体场景和构图问题给出鼓励性建议。
- **📸 一键拍照与相册**：按 **S** 键保存 1280×720 原图至 SD 卡；按 **L** 键进入相册，支持前后翻页（**A/D** 键）。
- **🔘 物理按钮支持**：6 个 GPIO 按键（拍照、相册、螺旋线、退出、上一张、下一张），完全脱离键盘鼠标。
- **🚀 开机自启动**：通过 systemd 用户服务，上电即进入全屏取景界面，无需任何人工操作。
- **🧵 多线程架构**：界面渲染与 AI 推理分离，主线程保持 30 FPS 流畅预览，后台每秒更新构图建议。

## 🧰 硬件需求

| 组件 | 规格 |
|------|------|
| 主控板 | 进迭时空 K1 MUSE Pi Pro（RISC‑V 64 位） |
| 摄像头 | MIPI CSI 接口（OV5640 等），系统识别为 `/dev/video20` |
| 显示屏 | 800×480 或 1024×600 小尺寸 LCD/HDMI 屏（本项目以 HDMI 小屏为主） |
| 存储 | MicroSD 卡（≥ 8 GB，用于保存模型和照片） |
| 散热 | 散热片 + 5 V 小型风扇（强烈推荐，保证长时间稳定运行） |
| 输入（可选） | 6 个轻触按键，连接至 K1 的 GPIO 70、71、72、73、91、92 |

## ⚙️ 软件依赖

- **操作系统**：Bianbu Linux（基于 Debian）
